## 5-2. 공통원소 구하기

### 문제

A, B 두 개의 집합이 주어지면 두 집합의 공통 원소를 추출하여 오름차순으로 출력하는 프로
그램을 작성하세요.

> 입력예제

- 13952
- 32578

> 출력예제

- 235

### 해결

단순히 해결하는 것 뿐만 아니라, 효율적인 알고리즘에 대해 공부하고 있다.

시간복잡도를 적게 가져가야한다는 의미이기도 하다.

- 2중 for문의 시간 복잡도 : O(n^2)
- sort의 시간복잡도 : O(nlogn)
- 투포인터 알고리즘 : O(n)

> 2중 for문을 사용한 해결

```jsx
<script>
      function solution(arr1, arr2) {
        let answer = [];
        const n = arr1.length;
        const m = arr2.length;
        for (let i = 0; i < n; i++) {
          for (let j = 0; j < m; j++) {
            if (arr1[i] === arr2[j]) answer.push(arr1[i]);
          }
        }
        answer.sort((a, b) => a - b);
        return answer;
      }

      let a = [1, 3, 9, 5, 2];
      let b = [3, 2, 5, 7, 8];
      console.log(solution(a, b));
    </script>
```

> 정렬한 후, 투포인터 알고리즘으로 해결

```jsx
<script>
      function solution(arr1, arr2) {
        let answer = [];
        let p1 = 0;
        let p2 = 0;
        const n = arr1.length;
        const m = arr2.length;
        arr1.sort((a, b) => a - b);
        arr2.sort((a, b) => a - b);

        while (p1 < n && p2 < m) {
          if (arr1[p1] === arr2[p2]) {
            answer.push(arr1[p1]);
            p1++;
            p2++;
          } else if (arr1[p1] < arr2[p2]) p1++;
          else p2++;
        }

        return answer;
      }

      let a = [1, 3, 9, 5, 2];
      let b = [3, 2, 5, 7, 8];
      console.log(solution(a, b));
    </script>
```
